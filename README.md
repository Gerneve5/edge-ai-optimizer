# Edge AI Optimizer

![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20macOS-lightgrey) ![Python](https://img.shields.io/badge/python-3.8%2B-blue)

Toolkit for optimizing deep learning models for deployment on edge devices.

## Features
- Model quantization (INT8, FP16)
- Pruning and sparsity optimization
- Knowledge distillation framework
